# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a **Claude Code 系统性学习路线图** — a structured 7-stage curriculum for learning Claude Code from scratch to advanced usage. All content is in Chinese.

## Project Structure

```
claude-code-learning/
├── README.md                     # 总览 + 7阶段索引 + 学习路径图 + 推荐资源
├── stage-01-basics.md            # 阶段一：基础操作与日常使用
├── stage-02-claude-md.md         # 阶段二：项目管理与 CLAUDE.md
├── stage-03-advanced-workflow.md # 阶段三：高级交互与工作流
├── stage-04-mcp.md               # 阶段四：MCP 生态集成
├── stage-05-skills-hooks.md      # 阶段五：Skills 与 Hooks 开发
├── stage-06-harness-engineering.md # 阶段六：深度定制与工作流工程
├── stage-07-production.md        # 阶段七：高级模式与生产化
```

Each stage file is self-contained with learning objectives, checklists, and evaluation criteria.

## How to Use

- Learners progress through stages sequentially (1 → 7)
- Each stage file contains the full learning content for that stage
- README.md serves as the navigation hub with difficulty ratings and time estimates
- The `.claude/settings.local.json` currently only has WebSearch permission allowed

## Key Commands

- View any stage: read `claude-code-learning/stage-XX-*.md`
- Resource links are at the bottom of README.md (official docs, community repos, Chinese guides)
