## 学习路径总览

```
第一周 ─ 基础操作 ─────────── 能当日常编码助手用
   ↓
第二周 ─ CLAUDE.md/项目管理 ─── 项目配置化，开箱即用
   ↓
第三周 ─ 高级交互/工作流 ──── 灵活选用交互策略
   ↓
第四周 ─ MCP 生态集成 ─────── 接入外部系统和数据
   ↓
第五周 ─ Skills + Hooks ──── 自建扩展和自动化
   ↓
第六周 ─ 深度定制与工程化 ─── 设计完整开发工作流
   ↓
第七周+ ─ 高级模式/生产化 ─── 成本优化、生态贡献
```
